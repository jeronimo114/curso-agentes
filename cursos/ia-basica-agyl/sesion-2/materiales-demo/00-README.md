## Antes de la sesión (15 min)

1. Imprimir `08-informe-regulatorio.html` a PDF (Cmd+P → Save as PDF) y guardar en esta misma carpeta como `08-informe-regulatorio.pdf`.
2. Imprimir cada uno de los 3 archivos en `notebooklm-fuentes/*.html` a PDF también.
3. Abrir esta carpeta en Finder, ventana visible.
4. Tener `prompts-demo.md` abierto en otra ventana lateral (al lado de Finder).
5. Tener listas las pestañas: Claude (Sonnet 4.5), ChatGPT, NotebookLM.
